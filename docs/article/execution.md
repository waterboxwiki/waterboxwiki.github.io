# 水箱等价执行目录

对于一些剧情，其对应的执行语句有很多种，而攻略/文案由于简略、游玩局限性的原因，不会列出所有的等价执行内容，故在此列出全部的等价执行内容。

**方位执行单独列出，不放在剧情里面。**

??? tip "第一次阅读须知"

    由于版本的特殊性，部分语句可能无法等价执行。
    
    鉴于精力有限，本目录不会进行二次验证，如有错误联系晨曦。

??? note "通俗文案标注说明"

    在同一行内，所有语句的执行效果相同，即等价执行。

??? abstract "历史版本"
